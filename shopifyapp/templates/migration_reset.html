Migration Reset Instructions

  1. Open your terminal or command prompt.
  2. Navigate to your Django project's root directory.
  3. Delete all files in the shopifyapp/migrations folder except __init__.py.
  4. Delete your database file (usually db.sqlite3).
  5. Run the following command: 
    python manage.py makemigrations shopifyapp
  6. Then run: 
    python manage.py migrate
  7. If you have any initial data or fixtures, load them now.
  8. Restart your Django development server.
Note: This process will reset your database. Make sure you have backups of any important data before proceeding.
